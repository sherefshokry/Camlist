# Camlist
Camlist iOS Engineer position's task using Clean Architecture Concepts with MVVM

### Installation

Dependencies in this project are provided via Cocoapods. Please install all dependecies with

`
pod install
`

## Model Specs

### Venues

| Property      | Type                |
|---------------|---------------------|
| `id`          | `String`            |
| `name`        | `String`            |
| `location`    | `Location`          |
| `categories`  | `[Category]`        |

### Location 

| Property           | Type               |
|--------------------|--------------------|
| `lat`              | `Double`           |
| `lng`              | `Double`           |
| `distance`         | `Int`              |
| `country`          | `String`           |
| `state`            | `String`           |
| `city`             | `String`           |
| `formattedAddress` | `[String]`         |


### Category

| Property      | Type                |
|---------------|---------------------|
| `id`          | `String`            |
| `name`        | `String`            |
| `pluralName`  | `String`            |
| `shortName`   | `String`            |

### Place Photo

| Property      | Type                |
|---------------|---------------------|
| `id`          | `String`            |
| `prefix`      | `String`            |
| `suffix`      | `String`            |
| `width`       | `Int`               |
| `height`      | `Int`               |
